# 🤖 RL Walker – Streamlit Reinforcement Learning Demo

A simple **Reinforcement Learning (RL)** project built with **Streamlit** that demonstrates how an agent learns to "walk" toward a goal using **Q-learning**.

---

## 🚀 Live Demo

Deploy easily using Streamlit Cloud.

---

## 🧠 Project Overview

This project simulates a basic environment where:
- An agent starts at position `0`
- The goal is to reach the final position
- The agent learns by trial and error using rewards

### 🎯 Objective
Teach an agent to move **right (towards goal)** efficiently.
